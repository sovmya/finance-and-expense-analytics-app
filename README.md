# Personal Finance & Expense Analytics App

A React-based finance tracker that helps users:
- Track income & expenses
- Analyze spending patterns
- Set budgets
- View financial insights through charts

## Features
- Add/Edit/Delete transactions
- Search, filter, sort
- Budget tracking with progress
- Analytics dashboard (charts)
- Recurring expenses
- Currency conversion

## Tech Stack
- React
- Context API
- Recharts
- React Hook Form + Yup
- Axios

## Run Locally
```bash
npm install
npm run dev
